# 文書ファイルの命名規則

このフォルダには数学まとめプリントの Markdown ファイルを保存します。
ファイル名は次の形式に従ってください。

```
年度-分野-テスト番号.md
```

- **年度**: 例 `R7`。
- **分野**: `A` = 数A、`B` = 数B、`I` = 数Ⅰ、`II` = 数Ⅱ、`III` = 数Ⅲ など。
- **テスト番号**: 下記の通り。
  1. 1学期中間
  2. 1学期末
  3. 2学期中間
  4. 2学期末
  5. 学年末

例: `R7-A-5.md` は「R7年度 数A 学年末」という意味です。

ドキュメント一覧は `docs/files.json` に自動的に出力されます。
ファイルを追加・削除したときは、ルートで

```
node scripts/generate_docs_index.js
```

を実行して最新の一覧を生成してください。
